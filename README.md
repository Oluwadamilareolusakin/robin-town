# Robin Town

Where Fashion Lives

# What it does
It is the Frontend for Robin Town, a Clothing Retail House, where fashion lives.

## Built With
- React JS
- ES6
- SCSS
- Webpack
#### and deployed to Heroku

## Live Demo

[Live Demo](https://robin-town.herokuapp.com)

## Getting Started

To get a local copy up and running follow these simple example steps.
- Clone the project unto your local machine
- `cd` into the project directory
- Run `npm i`
- Run `npm start`
- Open up `localhost:3000` in your browser to view the project
